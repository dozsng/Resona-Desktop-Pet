# 🌟 核心特性 (Core Features)

ResonaDesktopPet 不仅仅是一个简单的桌面挂件，它是一个集成了多种 AI 技术的深度交互系统。

## 1. 深度对话交互
- **多模型支持**：支持 OpenAI (DeepSeek/GPT-4), Google Gemini, Anthropic Claude 等多种大模型。
- **持久化记忆**：具备完整的对话历史管理，能够结合上下文进行连贯对话。
- **环境感知**：LLM 能够获取当前时间、日期等上下文信息，提供更智能的回复。

## 2. 全语音交互体验
- **高质量 TTS**：集成 GPT-SoVITS 推理引擎，支持多种情感表达，语音自然生动。
- **离线 STT**：基于 SenseVoice 的离线语音识别，保护隐私且响应迅速，支持通过快捷键呼出对话。

## 3. 极具个性的触发系统
- **环境触发器**：可以根据系统温度、硬件占用、当前运行的软件、浏览器访问的 URL 等自动触发反馈。
- **交互触发器**：支持鼠标悬停、长按、点击连击等多种交互方式的反馈配置。
- **自动化动作**：触发后可以执行一系列动作，包括播放台词、移动位置、改变透明度、随机动作分支等。

## 4. 灵活的资源包系统
- **一键切换**：支持完整的角色包切换，包括立绘素材、语音模型、人格提示词、触发逻辑等。
- **高度可定制**：通过简单的 JSON 配置即可定义角色的所有行为逻辑。

## 5. 完善的开发辅助工具
- **可视化编辑器**：提供图形化的触发器编辑器，无需手动编写复杂的 JSON。
- **模拟调试环境**：内置传感器模拟器，方便在不满足真实条件的情况下测试触发逻辑。
- **素材处理流水线**：自动化的立绘处理与组织工具，快速完成角色包制作。
