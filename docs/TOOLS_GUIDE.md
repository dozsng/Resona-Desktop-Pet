# 🛠️ 开发工具指南 (Tools Guide)

项目在 `tools/` 目录下提供了一系列辅助工具，帮助您快速制作和调试资源包。

## 1. 触发器编辑器 (`trigger_editor.py`)
**这是最核心的工具**，用于图形化编辑 `triggers.json`。
- **如何使用**：
  1. 运行 `trigger_editor.py`。
  2. 在左上角选择你要编辑的资源包。
  3. 点击“新增触发器”并设置 ID。
  4. **添加条件 (Conditions)**：如 CPU 温度 > 80 或 鼠标悬停 > 5 秒。
  5. **添加动作 (Actions)**：如 `speak` (语音回复) 或 `move_to` (移动位置)。
  6. 点击“保存同步”。
- **进阶**：支持复杂的逻辑组合（AND/OR），让宠物能够做出更智能的反应。

## 2. 传感器模拟器 (`sensor_mocker.py`)
用于调试触发器的利器。
- **用途**：当你在编写“CPU 高温警告”或“全屏模式反馈”等逻辑时，不需要真的让电脑变烫。
- **如何使用**：
  1. 在 `config.cfg` 中设置 `debugtrigger = true`。
  2. 运行 `sensor_mocker.py`。
  3. 在模拟器界面拖动滑块（如将 CPU 温度拉到 90）。
  4. 宠物会实时接收到这些模拟数据并尝试匹配触发器。

## 3. 立绘预处理器 (`image_processor.py`)
- **用途**：Resona 的 UI 渲染基于 1280x720 的画布。如果你的原始立绘尺寸不一，这个工具可以将它们自动缩放并对齐到画布的**正下方**。
- **如何使用**：将图片或包含图片的文件夹直接拖到脚本上运行。

## 4. 动画序列整理器 (`sprite_organizer.py`)
- **用途**：将零散的立绘文件批量重命名为规范格式，并根据你分配的情感自动生成 `sum.json`。
- **如何使用**：
  1. 打开包含立绘的文件夹。
  2. 为每张图片选择对应的情感标签。
  3. 输入服装 ID。
  4. 点击生成，工具会自动将文件复制到对应的资源包目录并创建索引。
